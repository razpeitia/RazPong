# RazPong

A C++ pong clone, made over the weekend just to learn C++ in the process.

![RazPong](https://github.com/razpeitia/RazPong/blob/master/PongScreenShot.png?raw=true)

# Keys

Q -> Quit  
Player 1: Up & Down  
Player 2: W & S

# Notes

Max Score: 10  
Once the max score reach the game just loops.
